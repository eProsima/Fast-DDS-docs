Generating *Fast DDS* source code with *Fast DDS-Gen*
=====================================================

This section explains how to generate the *Fast DDS* source code for a *Remote Procedure Calls* (RPC) application
using the *Fast DDS-Gen* tool and discuss the main methods of the generated code in two different situations:

* **Basic example**: A simple calculator service with basic asynchronous operations.
* **Advanced example**: A calculator service with data streaming operations.
